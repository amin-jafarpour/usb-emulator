
### Control 
- initial setup
- SETUP token
- SETUP data packet 
- IN token 
- out token 
- DATA0 packet
- DATA packet
### Bulk Transfer 
- large transfer, e.g. mass storage
- DATAx and ACK/NAK/STALL/NO ACK handshaking.
### Interrupt Transfer
- Small transfer, e.g., keybords and mice
- poll-driven
### Isochronous
- Time-critical streaming data, e.g., webcams and audio.
